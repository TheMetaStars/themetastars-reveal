# TheMetaStars

Provably fair reveal of NFT's metadatas.

## Installation

Clone the repo & install dependencies.

```bash
git clone https://github.com/TheMetaStars/themetastars-reveal/
cd themetastars-reveal
npm install 
```
Shuffle metadatas 
```bash
node index.js
```

## About

staticSeed is public : "themetastars"
